## API

| 参数    | 说明     | 类型      | 默认值 | 版本  |
| :------ | :------- | :-------- | :----- | ----- |
| show    | 是否展示 | `boolean` | false  | 0.0.1 |
| integer | 是否整数 | `boolean` | false  | 0.0.1 |
| dragWrapperClassName | 拖动区域的元素className | `string` | 'App'  | 0.0.1 |
| onClose | 点击关闭按钮的回调 | `function` | -  | 0.0.1 |
| onChange| 点击键盘数字按钮的回调 | `function` | -  | 0.0.1 |
